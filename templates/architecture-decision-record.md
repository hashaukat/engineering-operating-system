## Context

What is the situation that forced this decision? What constraints apply? What are we trying to optimize for? 2–4 paragraphs.

Include:
- Customer or business pressure driving the decision
- Technical constraints (existing systems, SLAs, compliance)
- Which [strategic bet](../docs/engineering-strategy.md) and horizon this advances
- Which [operating principles](../docs/operating-principles.md) are relevant

## Options considered

For each option, the same structure:

### Option A: `<short name>`
- **Summary** — 1–2 sentences
- **Pros** — bullet list
- **Cons** — bullet list
- **Cost** — money, time, complexity
- **Reversibility** — easy / hard / one-way door

### Option B: `<short name>`
(same structure)

### Option C: `<short name>` *(if generated by AI, mark it)*
(same structure)

> **AI usage note:** if any option was AI-generated, mark it. If any option was *rejected* based on AI critique, record the reasoning. This builds organizational memory per the [AI policy](../docs/ai-assisted-development-policy.md).
